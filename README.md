In order to run this project you need to:

### Setup

`git clone git remote add origin https://github.com/ClintonYounge/My-Bookstore-Project`

`git clone git remote add origin https://github.com/ClintonYounge/My-Bookstore-Project` <-- _Only if you have [SSH keys configured](https://docs.github.com/en/authentication/connecting-to-github-with-ssh/adding-a-new-ssh-key-to-your-github-account) on your GitHub account_.
